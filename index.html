Overview

3D perception of dynamic deformable objects is a fundamental problem that has broad applications in industrial, household, and surgical settings. Deformable objects can take an infinite variety of configurations that make it difficult to perceive their semantically meaningful parts. While geometric deep learning techniques that model local structures have shown success for rigid point cloud processing, they fall short for highly deformable objects. These deformable objects experience continuous transformations to their local geometric structure, particularly the immediate spatial relationships between features such as corners, edges, and planes. Topological data analysis (TDA) can model the global connectivity structure of an object, which remains stable under deformations. We propose to improve 3D perception of deformable objects by fusing topological and geometric information. We leverage the Mapper algorithm, a tool from TDA for modeling global connectivity by representing the topological structure of data as a graph. We extend the Mapper algorithm to operate on time-varying deformations with the Sequential Mapper algorithm. We also propose the TopoMetric Graph Neural Network (TOM GNN) for learning topological and geometric features to perform common point cloud processing tasks, including semantic part segmentation and classification. We benchmark this technique for 3D semantic part segmentation on our novel dataset called FabricDeform3D, which includes 75,000 labelled point clouds with 65 semantic part labels across 15 highly deformable object classes of garments, bags, and backpacks. Compared to well-known point cloud processing backbones, TOM GNN outperforms the strongest baseline by 1.9% with the mean IoU metric on FabricDeform3D. TOM GNN also generalizes to standard benchmarks with rigid point clouds like ShapeNet-Part and ModelNet40.
